# Chinese
``` shell
我们的足球队已经完成了冠军。

我们团队的比赛结果记录在字符串集合中。每场比赛都由以下格式的字符串表示，其中是我们球队的得分，是我们的对手得分。"x:y"xy

例如：["3:1", "2:2", "0:1", ...]

每场比赛的积分如下：

如果 X > Y：3 分（获胜）
如果 x < y：0 分（损失）
如果 x = y：1 分（平局）
我们需要编写一个函数来获取这个集合，并返回我们的团队 （） 根据上面给出的规则在锦标赛中获得的积分。x

笔记：

我们的球队总是在锦标赛中打 10 场比赛
0 <= x <= 4
0 <= y <= 4
```
